# **App Name**: CorpConnect Invite

## Core Features:

- Dynamic Invitation Generation: Generate unique, trackable invitation links for each invitee.
- Mobile-Optimized Design: Ensure a responsive and visually appealing design on all mobile devices.
- RSVP Management: Allow invitees to RSVP directly through the invitation.
- Automated Reminders: Send automated reminders to invitees who have not yet RSVP'd.
- Generative AI Design Assistant: AI tool will analyze the event details and user preferences to generate design drafts.
- Guest List Management: Provide a dashboard to manage the guest list, track RSVPs, and send updates.

## Style Guidelines:

- Primary color: Deep Purple (#673AB7) for sophistication and exclusivity.
- Background color: Light Gold (#F0E68C) to provide a luxurious and warm feel.
- Accent color: Gold (#FFD700) to highlight key elements and add a touch of opulence.
- Body and headline font: 'Inter', a sans-serif font, known for its clean and readable appearance on screens.
- Use modern, minimalist icons to represent different invitation actions (RSVP, location, etc.).
- Employ a clean, single-column layout optimized for mobile viewing. Use white space effectively to prevent clutter.
- Subtle animations on button hovers and page transitions to enhance user engagement.
